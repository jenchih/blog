## 技术栈介绍
1. 前端主要技术 ==vue2== ==webpack2== ==es6== 
2. 服务端主要技术 ==thinkphp5== ==swoole== ==hprose==

## 整体架构
- 博客前台展示页面数据由服务端的swoole和hprose的远程数据服务提供数据
- 博客后台管理页面数据由服务端的thinkphp5 作为restful api提供接口数据服务
